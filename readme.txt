== Description ==

This is a simple plugin that detects Google Adwords Traffic and keeps a database of IP's. A report is generated that shows any IP address that has been used more than 2 times. 

1. Several CSV reports available.
2. Archive button to clear ip's that can show up in report

It is always a good idea to run IP's through ARIN to see where they are from before you exclude them from you PPC program.

== Installation ==


1. Upload `ppc_fraud.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
